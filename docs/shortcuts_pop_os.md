#Shortcuts I use in Pop OS!

## Move windows around

| Handling                     | Tastaturgenvej          |
|-------------------------------|--------------------------|
| Move to top right corner      | Alt+Super+KP 9               |
| Move to top left corner       | Alt+Super+KP 7               |
| Move to bottom right corner   | Alt+Super+KP 3               |
| Move to bottom left corner    | Alt+Super+KP 1               |
| Move to top                   | Alt+Super+KP 8               |
| Move to right                 | Alt+Super+KP 6               |
| Move to bottom                | Alt+Super+KP 2               |
| Move to left                  | Alt+Super+KP 4               |
| Move to center/maximize       | Alt+Super+KP 5               |
| Move to configured location   | Alt+Super+KP 0               |
| Move to the right screen      | Shift+Super+Right        |
| Move to the left screen       | Shift+Super+Left         |
