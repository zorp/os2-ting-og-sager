#Shortcuts I use in Pop OS!

## Move windows around

| Handling                     | Tastaturgenvej          |
|-------------------------------|--------------------------|
| Move to top right corner      | Super+KP 9               |
| Move to top left corner       | Super+KP 7               |
| Move to bottom right corner   | Super+KP 3               |
| Move to bottom left corner    | Super+KP 1               |
| Move to top                   | Super+KP 8               |
| Move to right                 | Super+KP 6               |
| Move to bottom                | Super+KP 2               |
| Move to left                  | Super+KP 4               |
| Move to center/maximize       | Super+KP 5               |
| Move to configured location   | Super+KP 0               |
| Move to the right screen      | Shift+Super+Right        |
| Move to the left screen       | Shift+Super+Left         |
